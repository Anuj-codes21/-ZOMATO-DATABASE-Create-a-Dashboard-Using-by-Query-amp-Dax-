#  Zomato Data Analysis Dashboard (Power BI)

##  Project Overview
This project presents an end-to-end data analysis and business intelligence solution using Microsoft Power BI.  
The objective is to transform raw Zomato restaurant data into actionable insights through data cleaning, modeling, DAX calculations, and interactive visualizations.

The dashboard enables stakeholders to analyze restaurant performance, customer trends, ratings distribution, and geographical insights for data-driven decision-making.

---

##  Business Objectives
- Analyze restaurant distribution across cities and countries  
- Evaluate ratings and customer engagement trends  
- Identify top-performing cuisines and restaurant categories  
- Monitor cost distribution and pricing patterns  
- Generate actionable KPIs for business growth  

---

##  Tools & Technologies Used
- **Microsoft Power BI**
- **Power Query** (Data Cleaning & Transformation)
- **DAX (Data Analysis Expressions)**
- Data Modeling (Relationships & Calculated Measures)
- Excel / CSV Dataset

---

##  Dashboard Features
- Interactive and user-friendly Power BI dashboard  
- KPI Cards for quick business insights  
- Dynamic filters and slicers  
- City-wise and country-wise performance analysis  
- Ratings and cost distribution visualization  
- Cuisine-based performance comparison  
- Drill-down and drill-through functionality
